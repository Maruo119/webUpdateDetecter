- **パターン1（1件差分）**: 全6社で正確に1件を検知

差分検知アルゴリズムの仕組み：
```python
prev_hrefs = set(state_before[site_name])
```

**差分検知ロジックは汎用的で信頼性が高い** ことが確認されました。
